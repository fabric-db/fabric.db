# Dot Grammar.

Every statement must end with a full stop.

The full stop is the statement terminator.

It closes the statement.

It seals the meaning.

It preserves the boundary.

## Rule.

```text
statement.
```

A fabric statement is not complete until it ends with a full stop.

## Examples.

```text
Box is the building block of fabric.

Fabric is a structured multimodel graph database.

Agreement is temporal.

Data is defined, not fabricated.

Zero loss. Zero trust. Zero touch.
```

## Canonical law.

Every statement must end with a full stop.
